# sippr-app
JavaScript 2 Course - Social Media Application
![Rainydays-w-js project screenshot](https://raw.githubusercontent.com/tominehaug/portfolio/main/images/rainydays-js.jpeg)

Sippr is a social media app for sharing posts both with and wothout images, including interacting with other users through following. It is a project from an intermediate JavaScript course in the third semester of my Front-End Development studies at Noroff Vocational School.

## Description

The main focus of this solidify and expand my knowledge of Vanilla JS, as well as adding basic UI and design. 
The JavaScript includes following features:

- API calls
- Events
- Array manipulation
- Object destructuring
- ES6 Modules

The purpose with this project was to practice:
- Developing interactive and dynamic web solutions with JavaScript
- Data structure manipulation techniques
- Writing modular code using ES6 modules
- TypeScript and testing on an introductory level
- Implementing standalone applications in JavaScript

## Getting Started

### Installing

1. Clone the repo:

```bash
git clone https://github.com/tominehaug/sippr-app.git
```
Run this command in the Visual Studio Code Terminal.

2. Install the dependencies:

No dependencies for this project.

### Running

To run the app use LiveServer extension in Visual Studio Code.
